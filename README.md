1. 프로젝트 제목 및 소개

🛡️ Password Security Checker   
정보보안기사 학습 내용을 바탕으로 제작한 파이썬 기반 비밀번호 강도 검사기입니다.

2. 주요 기능 (Features)

강도 측정: 비밀번호 길이를 분석하여 점수 부여 (비교 연산자 활용)

패턴 검사: 숫자 및 특수문자(!@#$%^&*) 포함 여부 확인 (멤버십 연산자 활용)

실시간 피드백: 보안 강도에 따른 맞춤형 개선 가이드 출력 (Match-Case 활용)
